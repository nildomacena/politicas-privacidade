Política de Privacidade

Alagoas Preço Baixo

Esta política de privacidade descreve como o aplicativo Alagoas Preço Baixo lida com os dados dos usuários.

Coleta de Dados

O aplicativo coleta dados do usuário exclusivamente para exibição de anúncios. Não é realizado nenhum tipo de cadastro ou armazenamento permanente de informações pessoais.

Uso dos Dados

Os dados coletados são processados de maneira efêmera, apenas durante o uso do aplicativo, e não são compartilhados com nenhuma outra empresa ou terceiros.

Compartilhamento de Dados

Não compartilhamos dados dos usuários com nenhuma empresa, organização ou pessoa externa.

Segurança

Tomamos medidas para garantir que os dados coletados sejam utilizados apenas para a finalidade de exibição de anúncios e não sejam armazenados após o uso.

Contato

Em caso de dúvidas sobre esta política, entre em contato pelo e-mail: koinedevelopment@gmail.com
