Privacy Policy - Med Docs

English

The Med Docs application is designed to store medical test results. The app requires user login to access its features.

Users can save their medical data and test results within the application. This data is stored securely and is not shared with any third parties.

The app only shares data for the purpose of displaying advertisements, following Google AdMob's standard data sharing practices.

No personal or medical information is required beyond login credentials, and all stored data remains private to the user.

For any questions or additional information, please contact us by email: koinedevelopment@gmail.com.
