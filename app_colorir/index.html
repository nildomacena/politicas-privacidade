Política de Privacidade

App de Colorir

Introdução

Bem-vindo à Política de Privacidade do App de Colorir. Esta política descreve como tratamos suas informações ao usar nosso aplicativo. O App de Colorir está comprometido em proteger sua privacidade e garantir uma experiência segura e divertida para todos os usuários.

Informações que Coletamos

O App de Colorir coleta um conjunto muito limitado de informações:

  • ID do Dispositivo: Coletamos apenas o identificador do dispositivo utilizado. Esta é a única informação pessoal que obtemos.
  • Dados de Uso: Informações básicas sobre como o aplicativo é utilizado, de forma anônima e agregada.

Como Usamos as Informações

O ID do dispositivo coletado é utilizado exclusivamente para:

  • Exibição de anúncios personalizados
  • Melhorar a experiência do usuário
  • Solucionar problemas técnicos

Não utilizamos essas informações para rastrear individualmente os usuários ou criar perfis pessoais detalhados.

Compartilhamento de Dados

Não compartilhamos, vendemos ou alugamos seus dados pessoais a terceiros. As únicas exceções são:

  • Plataformas de Anúncios: Compartilhamos o ID do dispositivo com plataformas parceiras de anúncios para exibição de propagandas relevantes
  • Requisitos Legais: Quando exigido por lei ou para responder a processos legais

Segurança de Dados

Implementamos medidas técnicas e organizacionais para proteger as informações limitadas que coletamos. No entanto, nenhum sistema é completamente seguro, e não podemos garantir a segurança absoluta dos seus dados.

Direitos do Usuário

Você tem o direito de:

  • Acessar os dados limitados que temos sobre você
  • Solicitar a exclusão dos seus dados
  • Optar por não receber anúncios personalizados (através das configurações do seu dispositivo)

Uso por Crianças

O App de Colorir pode ser utilizado por crianças, mas não coletamos intencionalmente informações pessoais de menores de 13 anos além do ID do dispositivo necessário para a funcionalidade básica do aplicativo.

Alterações nesta Política

Podemos atualizar nossa Política de Privacidade ocasionalmente. Recomendamos verificar esta página periodicamente para quaisquer alterações. As mudanças entram em vigor imediatamente após serem publicadas.
