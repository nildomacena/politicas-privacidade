Política de Privacidade - Vasco App

Sobre o Aplicativo

Vasco App é um aplicativo centralizador de notícias sobre o Club de Regatas Vasco da Gama, oferecendo aos torcedores vascaínos as mais recentes informações sobre o clube.

1. Informações Gerais

Esta Política de Privacidade descreve como o Vasco App coleta, usa e protege as informações dos usuários. Ao utilizar nosso aplicativo, você concorda com os termos descritos nesta política.

2. Informações Coletadas

O Vasco App coleta apenas as informações necessárias para o funcionamento adequado do aplicativo e para a exibição de publicidade relevante.

2.1 Dados Coletados para Publicidade

  • Identificadores de dispositivo: ID do dispositivo, ID de publicidade
  • Informações de localização aproximada: País, região ou cidade (não localização precisa)
  • Dados de uso do aplicativo: Páginas visitadas, tempo de uso, interações com anúncios
  • Informações técnicas: Modelo do dispositivo, versão do sistema operacional, versão do aplicativo
Importante: O Vasco App NÃO coleta dados pessoais como nome, e-mail, telefone ou outras informações de identificação pessoal.

3. Finalidade da Coleta de Dados

Os dados coletados são utilizados exclusivamente para:

  • Exibição de anúncios relevantes e personalizados
  • Melhoria da experiência do usuário no aplicativo
  • Análise de desempenho e otimização do aplicativo
  • Cumprimento de obrigações legais quando aplicável

4. Compartilhamento de Dados

Os dados coletados podem ser compartilhados com:

  • Redes de publicidade: Para exibição de anúncios personalizados
  • Serviços de análise: Para melhorar o desempenho do aplicativo
  • Autoridades competentes: Quando exigido por lei

Não vendemos, alugamos ou comercializamos seus dados para terceiros para outros fins.

5. Armazenamento e Segurança

  • Os dados são armazenados em servidores seguros com criptografia
  • Implementamos medidas técnicas e organizacionais apropriadas para proteger os dados
  • Os dados são mantidos apenas pelo tempo necessário para cumprir as finalidades descritas

6. Direitos dos Usuários

Você tem os seguintes direitos em relação aos seus dados:

  • Acesso: Solicitar informações sobre os dados coletados
  • Correção: Solicitar correção de dados incorretos
  • Exclusão: Solicitar a exclusão dos seus dados
  • Portabilidade: Solicitar a transferência dos seus dados
  • Oposição: Opor-se ao processamento dos seus dados para publicidade

7. Cookies e Tecnologias Similares

O aplicativo pode utilizar cookies e tecnologias similares para:

  • Melhorar a funcionalidade do aplicativo
  • Personalizar a experiência do usuário
  • Coletar informações analíticas
  • Exibir publicidade relevante

8. Publicidade

O Vasco App exibe anúncios de terceiros que podem coletar informações através de suas próprias tecnologias. Recomendamos que você consulte as políticas de privacidade dos anunciantes para mais informações.

9. Alterações nesta Política

Esta Política de Privacidade pode ser atualizada periodicamente. Notificaremos os usuários sobre mudanças significativas através do próprio aplicativo. A versão mais recente estará sempre disponível nesta página.

10. Lei Aplicável

Esta política é regida pelas leis brasileiras, incluindo a Lei Geral de Proteção de Dados Pessoais (LGPD) - Lei nº 13.709/2018.
